Ejemplo del uso del interpolation binding. Cada instancia del componente lee un arreglo declarado en su controlador y muestra sus valores usando expresiones del tipo {{ opciones[0] }} y {{ opcionI }} con un *ngFor="let opcionI of opciones" .

seleccionador
configurable
Ejemplo de usos de directivas estructurales. En este caso:
  • *ngFor (with index),
  • *ngIf,
  • [ngSwitch] (no estructural),
  • *ngSwitchCase y
  • *ngSwitchDefault.
También se utiliza class binding para el dibujo condicional del borde.
El componente permite personalizar:
  1. Borde
  2. El despliegue de valores de cada opción
  3. Si se desea usar radio botones o botones de tipo chekbox
  4. El nombre del campo
  5. Los valores posibles a elegir
En general, el componente explota el renderizado condicional con base en datos residentes en su controlador.

Componentes que devuelven valores 
selec-alternativa

Valor Elegido:

{{valorPelicula}}

El componente de selección recibe objetos que provienen del componente (usando el decorador @Input()).
El componente publica los cambios de valor por medio del uso de un objeto emisor de eventos EventEmitter<string> y del decorador @Output.
Se le pasan objetos al seleccionador por medio de [campo]="campoPelicula", que indica el uso de una Input property llamada 'campo' (que tomará el valor del objeto local 'campoPelicula'). También se escucha el evento publicado por el componente. Ésto le permite recibir los valores devueltos por el componente (el seleccionador).
