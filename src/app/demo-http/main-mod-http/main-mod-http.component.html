. Usan el api HttpClient y tratan respuestas asíncronas.
Ejemplo de implementación de peticiones HTTP a un servidor. Se inyecta un servicio de datos que hace las peticiones y maneja los errores que provienen del servidor o de la red. El procesamiento asíncrono usa observables, que son manipulados por medio del API de RxJs.
El componente que despliega los mensajes de error, usa métodos del Lifecycle event sequence de Angular para manejar adecuadamente su estado.
